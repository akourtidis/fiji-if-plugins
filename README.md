# fiji-if-plugins
Plugins for quantification of junctional markers in immunofluorescence images

## Installation Instructions
1. Open FIJI
2. In the menu bar: Plugins → Install...
3. Open the IF_plugins.jar file
4. Restart FIJI

## To organize plugins: 
1. Navigate to your FIJI installation directory.
2. Open the plugins folder
3. Create a new subfolder inside the plugins folder (e.g., IF_plugins)
4. Move the IF_plugins.jar file into the new subdirectory
